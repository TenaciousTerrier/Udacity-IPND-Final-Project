Important Concepts

Lesson 1.1: The Basics of the Web and HTML

The World Wide Web:

The web is a network of computers that communicate with each other. When a user enters a URL like "www.facebook.com" or "www.amazon.com" into a web browser, an HTTP request is sent from their computer to a server.The server takes the URL as an address and retrieves the corresponding HTML document which is rendered via the user's web browser and displayed on the computer screen. You can find more information here: World Wide Web, Wikipedia

HTML: Hypertext Markup Language:

HTML is an abbreviation for "Hypertext Markup Language". Most content on the web is made up of HTML documents which can contain text, video, music, and many other file types. HTML files contain text content and markup. Text content describes what is shown on your computer screen, and markup determines how it is presented or visualized. More information can be found here: HTML, Wikipedia

Tags and Elements:

Elements and tags are what make up HTML documents. Elements define individual components of an HTML document, and tags define each element's type.

Are Computers "Stupid"?

Computers aren't actually "stupid", but they do take provided instructions literally. Even a small mistake in the computers instuctions, or code, oftentimes causes massive errors, even causing the failure of the entire set of instructions.

Inline and Block Elements:

HTML elements can be either inline or block elements. Block elements place invisible rectangles around their content whereas inline elements do not, and are not separated by other elements.

HTML Structure:

HTML possesses a "tree-like" structure. What does this mean? It means that HTML elements can contain other HTML elements. A good analogy is to think of a human nuclear family; parents have children. In HTML, the children would be child elements to the parents, who would be parent elements.

Boxes and Indentation:

Boxes and indentation are used to structure HTML documents. Indentation is used inside of block elements, and boxes are used create the structure of the web page.

Text Editors:

Programmers use text editors to assist with writing code. In this course, three text editors have been introduced: Scrathpad, CodePen, and Sublime Text. Sublime Text is the most difficult to use, but also has the most functionality. For this assignment, I created my HTML and CSS documents with the Sublime Text editor.

Lesson 1.3: Adding CSS Style and HTML Structure

Avoid Repetition Whenever Possible:

Avoiding repetition and eliminating redundancy as much as possible is critical to being an effective programmer. There are two main reasons for this:

Preventing Mistakes Repeating oneself lends to making mistakes. The more you copy and paste or rewrite the same code, the more likely you are to make a mistake that prevents you from achieving your programming goals. The less you repeat yourself, the less errors you are likely to make.

Being Consistent Consistency does a few things. Firstly, it helps your mind check your code and more easily spot errors. Secondly, it enables quick editing and make quick changes to style.

CSS:

CSS is an abbreviation for Cascading Style Sheets controls the style of an HTML document, or how is displayed in a web browser. To modify the style of an HTML document, similar HTML elements are assigned the identical class names, and a style is specified to that class, ensuring that all of these elements have the same style.
