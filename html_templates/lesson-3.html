Important Concepts

Lesson 3.1: Introduction to Abstraction

Utilizing the concept of abstraction, by using classes and functions inside of your programs, can greatly reduce repetition of code.

Lesson 3.2: Use Functions

Functions can be called, or executed, multiple times within code. This eliminates repetitive, and redundant blocks of code.

Lesson 3.3: Use Classes

  1. Class:
    • Blueprint of an object containing certain pieces of information about the object.
    • Blueprint can be used to create multiple different instances of an object.
  2. Instance
    • Refers to a specific class.

Lesson 3.4: Make Classes

  1. Constructors:
    • __init__ function
    • __init__ creates space in memory to remember details about a class.
    • Takes argument self, and other user-defined variables.
    • Initializes variables, also known as instance variables, associated with the class in which it is contained.
  2. Self
    • Object being created.
    • Not a Python keyword, but it is a convention used by most Python programmers.
  3. Instance Method
    • Function defined inside class and associated with an instance
